## üéÆ Overview

**Tubed** is a classic "Ball Sort" puzzle game where the objective is to sort colored balls into tubes until each tube contains only one color. It features a progressive difficulty system, sleek dark UI, and smooth animations.

## ‚ú® Features

- **40 Challenging Levels**: From beginner to expert difficulty.
- **Hidden Mechanics**: Levels 16+ introduce hidden balls, adding a layer of strategic depth.
- **Star System**: Earn up to 3 stars based on your efficiency (number of moves).
- **Persistent Progress**: Automatically saves your levels, stars, and best scores to local storage.
- **Responsive Design**: Optimized for mobile devices and desktops alike.
- **Modern UI**: Clean, dark aesthetic with vibrant colors and smooth transitions.

## üöÄ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/radekkubryn/Tubed.git
   cd Tubed
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
